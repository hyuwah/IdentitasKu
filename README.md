# IdentitasKu

An app to centralize your important info.

## Features

- [x] Common Indonesian Important Data
- [x] Copy to clipboard
- [x] Centralize info, no need to memorize it again
- [x] Password protection
- [x] Login with fingerprint
- [x] Encrypted db
